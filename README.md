# jscpd-bootstrap-reporter

Starter for jscpd reporter

## Getting started

### Get Starter
```bash
$ git clone https://github.com/kucherenko/jscpd-bootstrap-reporter.git 
```

### Install Dependencies
```bash
$ npm install
```

### Build Reporter
```bash
$ npm run build 
```

### Run Tests
```bash
$ npm test
```
with coverage
```bash
$ npm run cov
```
watch changes 
```bash
$ npm run watch
```

### Fix Code Styles 
```bash
$ npm run fix
```

### Prepare Release
```bash
$ npm run prepare-release
```
